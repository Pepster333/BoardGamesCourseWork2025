This is a step-by-step guide on how to operate the prototype

0: Download the rar file (ISProjectPrototype.rar)
1: Unzip the rar file
2: Setting up the local database inside visual studio:
2.1: From the menu View open SQL Server Object Explorer
2.2: Create a database and set a name for it 
2.3: Copy the ConnectionString from the properties menu
3: Change the ConnectionString inside appsettings.json
4. Run the program.
